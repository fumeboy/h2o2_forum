# forum-app

- Django REST Framework
- Vue.js
- Vue Router
- Vuex
- Axios
- Bulma

how to start it:

backend:

```
python3 manage.py makemigrations

python3 manage.py migrate 

python3 manage.py runserver
```

frontend:

```
yarn

yarn mock
```


Additionally, this is a way to test my django-kit's configuration.
